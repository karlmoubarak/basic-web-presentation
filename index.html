One HTML file with a small JS snippet and built-in CSS styling.

Instructions

  1. Download basic-web-presentation.zip.
  2. Put your images and other media files in the "src" folder.
  3. Edit the "index.html" file to your liking.

Document Structure

All "slides" are <section> tags nested in a <main> tag. Use left and right arrow keys to navigate. This will add and remove a "hidden" class to all sections sequentially.
